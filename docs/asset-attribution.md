# Asset Attribution

## AI Town Character Sprites

OpenNetwork currently uses the inherited AI Town character sprite sheet at
`public/assets/32x32folk.png` for event-town avatars and default town characters.
The surrounding project is distributed under the MIT license in `LICENSE`.

If these sprites are replaced with a third-party character pack, record the pack
name, author, source URL, license, and required attribution text here before
shipping it.
